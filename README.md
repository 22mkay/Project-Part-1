## Project Overview
This website is an online sneaker retailer that sells high-quality and high-end sneakers. It is built for sneakerheads and collectors, offering highly exclusive footwear at an affordable price. DraFT.sneakers is designed to be more than an online store — it’s a digital sneaker hub.

## Key Features & Functionality
* **Responsive Design:** Adapts fluidly across desktop, tablet, and mobile orientations.
* **Dynamic Search & Filter:** Quick access to desired sneakers using real-time JavaScript filtering.
* **Interactive Product Gallery:** Clean image display with lightbox capabilities for detailed viewing.
* **Shopping Cart State:** Functional "Add-to-cart" buttons with a live, updating navigation badge.
* **Optimized Codebase:** SEO-friendly HTML structure with semantic tags and clean CSS styling.
* **Payment Gateway Integration (Planned):** Support for secure checkouts.

## Website Analysis
* **Strengths:** Strong brand concept, clear product focus, recognizable name, and a clean, minimalist user interface.
* **Weaknesses:** E-commerce backend systems (database management) are still in the theoretical/planning stages.
* **Opportunities:** Build a vibrant platform that not only sells sneakers but also celebrates sneaker culture through visuals and community engagement.

## Technical Requirements
* **Frontend Languages:** HTML5, CSS3, Vanilla JavaScript (ES6) for custom DOM interactions.
* **Backend (Planned):** Node.js or PHP with MySQL/PostgreSQL database.
* **E-commerce Framework:** Custom-built cart system with future Shopify integration capability.
* **Hosting:** GitHub Pages / Netlify (for static deployment).
* **Security:** SSL certificate, secure login, and encrypted transactions.
* **Browser Support:** Chrome, Firefox, Safari, Edge.

## Design & User Experience
* **Colour Palette:** Red (`#ff3c00`), Black, White — bold, urban, and stylish.
* **Typography:** Clean sans-serif font for modern readability.
* **Layout:** Minimalist product grid with high-quality sneaker imagery.
* **Interactivity:** Hover animations, smooth CSS transitions, and instant visual feedback on buttons.
* **Brand Feel:** Energetic, street-inspired, and culturally connected.
